# positiverails-platform
Enterprise payment orchestration infrastructure and developer hub. POSitiveRails is a dynamic payload routing engine built natively on the NMI switchboard for high-velocity B2B, retail surcharging, and Web3 bifurcation. Operated by NDPS LLC.

POSitiveRails™ Infrastructure & Developer HubProprietary Orchestration Technology by New Digital Payment Systems (NDPS) LLC.OverviewPOSitiveRails is an enterprise-grade payment orchestration layer that operates exclusively on the data layer. By integrating directly into the NMI (Network Merchants Inc.) switchboard via the Advanced Transaction Routing Interface (ATRI), POSitiveRails dynamically evaluates JSON payloads in milliseconds.We insulate ISVs and mid-market merchants from aggregator risk (Stripe, Square, PayPal) by assigning dedicated acquiring endpoints based on the specific risk profile and profitability of the payload.Core Architectural CapabilitiesB2B Commercial Optimization: Automated injection of Level 2 and Level 3 data for freight, logistics, and corporate purchasing cards.Compliant Retail Surcharging: Terminal-level orchestration of dual-pricing logic (RewardPay) to achieve zero-effective rate processing.Fiat & Web3 Bifurcation: Parallel routing protocols that process credit cards via traditional TSYS/Fiserv rails (Priority/PayRoc) while simultaneously routing Stablecoins and APMs via Nuvei Digital APIs.Ecosystem Positioning (Node 2)This repository contains the front-end infrastructure for positiverails.com, serving as the Developer Hub and Institutional Partner portal.It is "Node 2" of the NDPS Ecosystem:Node 1 (omni-pos.platform): Merchant-facing UI / Point of Sale software.Node 2 (positiverails.com): The underlying routing engine and partner API documentation.Node 3 (NDPS LLC): Master ISO legal entity and institutional vault.Tech StackFramework: HTML5 / Vanilla JSStyling: Tailwind CSS (via CDN)Design Language: Deepspace (#050515), Inter/Space Mono fonts, Glassmorphism UIDeployment: Cloudflare Edge Network / GitHub Pages© 2026 New Digital Payment Systems (NDPS) LLC.
